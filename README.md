# Client Video Recorder

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd my-video-recorder
   ```

2. Install dependencies and run the code:

   ```bash
   npm install
   npm run dev -- --open
   ```

## Packages used
